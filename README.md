# **Image Carousel Project**

## **Project Overview**
This project is an interactive **Image Slider / Carousel** built using **HTML, CSS, and JavaScript**.  
It displays a slideshow of images that automatically changes after a fixed time interval and allows users to navigate manually using **Previous** and **Next** buttons.

---

## **Features**
- Automatic slideshow using `setInterval()`
- Previous and Next navigation buttons
- Dot indicators for slide selection
- Responsive and attractive UI design
- Pure JavaScript implementation (no external libraries)

---

## **Technologies Used**
- **HTML5**
- **CSS3**
- **JavaScript**
